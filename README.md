# Week 2 – SOC Workflow Documentation

This folder contains all tasks completed for Week 2 of the CyArt SOC Internship.  
The focus of this week was on SOC analysis workflows, alert management, incident classification, and documentation skills.

## Included Files

- **Alert-Classification.xlsx**  
  Contains alert triage, MITRE mapping, and priority scoring.

- **Incident-Classification.pdf**  
  Categorization of incidents using MITRE ATT&CK, ENISA, and metadata fields.

- **Incident-Response-Lifecycle.pdf**  
  Summary of the 6 phases of Incident Response.

- **IOC-Validation-Report.pdf**  
  Report showing IOC checked in VirusTotal and validated.

- **Evidence-Preservation.pdf**  
  Documentation of chain-of-custody evidence handling.

- **Capstone-Final-Report.pdf**  
  Complete simulated incident: Attack → Detection → Response → Remediation → Recommendations.

## Summary
Week 2 focused on building SOC analyst skills through documentation, triage, and workflow simulation.  
All tasks have been completed as per instructions.
