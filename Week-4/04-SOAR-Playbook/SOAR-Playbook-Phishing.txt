Trigger:
Phishing or suspicious file download alert generated by Wazuh SIEM.

Playbook Steps:

Step 1: Indicator Extraction
- Extract source IP address and file hash from the alert.

Step 2: Reputation Check
- Automatically check IP and file hash reputation using VirusTotal.

Step 3: Automated Containment
- If malicious:
  - Block the source IP using CrowdSec.
  - Prevent further communication.

Step 4: Case Creation
- Create an incident case in TheHive.
- Attach alert details and actions taken.

Step 5: Analyst Notification
- Notify SOC analyst for review.

Outcome:
Automates validation, containment, and case creation
to reduce response time for phishing incidents.
