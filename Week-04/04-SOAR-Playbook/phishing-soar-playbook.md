\## Tool Reference

TheHive (design-only)



\## Objective

Automate the response to phishing alerts to reduce SOC response time.



\## Trigger

A phishing-related alert generated by Wazuh containing a suspicious IP or URL.



\## Inputs

\- Source IP address

\- URL



\## Automated Logic

1\. Extract IP and URL from the alert.

2\. Check reputation using threat intelligence (VirusTotal / AlienVault OTX).

3\. If malicious, block the IP and update the case.

4\. If clean, mark the alert as a false positive.



\## Case Closure

All actions and results are documented in TheHive and the case is marked as Resolved.
